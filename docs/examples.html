Quick Start Examples

Basic Cross-Chain Swap
Complete example of a cross-chain HTLC swap

This example demonstrates how to perform a complete cross-chain swap from Ethereum to Cosmos using the EVMore SDK.

import { FusionCosmosClient, EthereumHTLCClient } from '@evmore/sdk'; async function performCrossChainSwap() { // Initialize clients const cosmosClient = new FusionCosmosClient({ rpcUrl: 'https://rpc.osmosis.zone', chainId: 'osmosis-1' }); const ethereumClient = new EthereumHTLCClient({ rpcUrl: 'https://eth-mainnet.alchemyapi.io/v2/YOUR_KEY', contractAddress: '0x1234567890123456789012345678901234567890' }); // Create swap parameters const swapParams = { fromToken: 'ETH', toToken: 'ATOM', amount: '1000000000000000000', // 1 ETH (18 decimals) recipient: 'cosmos1qqqqqqqqqqqqqqqqqqqqqqqqqqqqqqqqnrql8a', timelock: 3600 // 1 hour }; // Create HTLC on Ethereum const htlcResult = await ethereumClient.createHTLC(swapParams); // Wait for confirmation await ethereumClient.waitForConfirmation(htlcResult.txHash); // Complete swap on Cosmos const swapResult = await cosmosClient.completeSwap({ htlcId: htlcResult.htlcId, secret: htlcResult.secret }); console.log('Swap completed:', swapResult); }
