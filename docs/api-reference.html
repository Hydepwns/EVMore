JavaScript/TypeScript SDK

EVMore SDK
Complete TypeScript SDK for cross-chain HTLC operations

The EVMore SDK provides a complete TypeScript interface for interacting with cross-chain HTLC contracts, managing swaps, and integrating with the EVMore protocol.

Installation

npm install @evmore/sdk # or yarn add @evmore/sdk

Quick Start

import { FusionCosmosClient, EthereumHTLCClient } from '@evmore/sdk'; // Initialize clients const cosmosClient = new FusionCosmosClient({ rpcUrl: 'https://rpc.osmosis.zone', chainId: 'osmosis-1' }); const ethereumClient = new EthereumHTLCClient({ rpcUrl: 'https://eth-mainnet.alchemyapi.io/v2/YOUR_KEY', contractAddress: '0x...' }); // Create a cross-chain swap const swap = await cosmosClient.createSwap({ fromToken: 'ATOM', toToken: 'ETH', amount: '1000000', // 1 ATOM (6 decimals) recipient: '0x742d35Cc6634C0532925a3b844Bc9e7595f06B09' });
